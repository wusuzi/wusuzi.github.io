css_doodle

Attributes - 属性

grid

The n x m follows the order of (col, row) or (x, y). The following formats of grid value are all recognizable:

  • grid = "0"
  • grid = "5"
  • grid = "20"
  • grid = "5x7"
  • grid = "5 x 7"
  • grid = "5,7"

There’s an alternative way to set up the grid by using the @grid property.

use

从css变量导入规则

seed

All random values will be regenerated to the same value as last time based on the seed value. It’s quite useful if you want to keep a snapshot.

所有随机值将根据种子值重新生成为与上次相同的值。如果你想保留一个快照，它是非常有用的。

Selectors - 选择器

:doodle

表示当前这个css-doodle的根元素 指向组件本身

:container

container是包含所有单元格的容器元素，它使用Grid Layout。你可能想在里面设置网格间隙。

会继承:doodle的属性

@nth(n, …)

类似于css选择器的nth-child(n)

@even

类似于css选择器的:nth-child(even) 选择双数

@odd

类似于css选择器的:nth-child(odd) 选择单数

@at(col, row)

在特定的列和行上选择单元格。

@random([ ratio ])

随机选择单元格 radio表示选中的概率 默认0.5 期待值0-1

@row(n, …)

选择某一行 – > 支持even双数和odd单数选择行

@col(n, …)

选择某一列 同上

@match(expression)

基于一般数学表达式定位单元格。变量i, I, x, X, y和Y可以不带前缀@符号用于计算

i => 格子

I => 总格子

x，y x轴 y轴

X，Y x轴总长 y轴总长

Properties

@place

等同于定位

@shape

各种形状

https://css-doodle.com/#property-@shape

Functions

如果函数没有形参，括号可以省略。

@index 坐标 简写 @i

@row 行 @y

@col 列 @x

@size-row 行宽 @X

@size-col 列高 @Y

@size 格子数量 @I

@pick 随机选择 @p

@pick-n(v1, v2,…) 顺序随机选择 @pn

@pick-d(v1, v2,…) 随机选择 随机有规律 @pd

@rand(start [,end]) 两数之间随机值 @r

@repeat(times, value) 重复一个值数次 @rep
